## Geopolitica / Macro
# (ejemplos — reemplazar con tus observaciones reales)
# - Guerra o conflicto entre paises grandes -> riesgo elevado -> SKIP por 4-6h
# - Sube tasa de interes Fed -> mercado baja -> SKIP por 24h
# - Colapso de banco o exchange grande -> primeras 2h SKIP, luego puede rebotar

## Bitcoin
# - BTC cae mas de 3% en 1h -> no comprar altcoins (correlacion alta)
# - BTC sube mas de 5% en 1h -> buena ventana para altcoins BNB
# - BTC dominancia >60% -> altcoins sufren, ser mas selectivo

## Hacks / Seguridad
# - Hack a un protocolo DeFi -> SKIP tokens del mismo sector por 12h
# - Hack a un exchange CEX grande -> SKIP todo por 4h, monitorear stables

## KOLs
# (agregar cuando observes patrones reales)
# - Tweet de CZ o Binance oficial sobre token BSC -> pump rapido, salida en <2h
# - Multiples KOLs diferentes hablando del mismo token el mismo dia -> mas confiable

## Señales falsas comunes
# - Pump de meme sin volumen sostenido (dura <30 min) -> ignorar
# - Noticia positiva pero precio ya subio >20% -> demasiado tarde para entrar

## Stablecoins
# - Dominancia stablecoins >15% -> mucha gente fuera del mercado, cautela
# - Dominancia stablecoins <7% -> capital rotando activamente, oportunidad
